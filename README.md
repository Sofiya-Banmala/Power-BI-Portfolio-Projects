# Power-BI-Portfolio-Projects

## **Project Overview**

This is a Power BI portfolio project and the goal of this project is to develop a Power BI dashboard that provides insights into student performance based on various academic and behavioral factors. This will help educators and administrators track progress, identify at-risk students, and improve decision-making.

## **Features**

This dashboard offers a comprehensive view of student performance through various key metrics. It tracks overall student performance which includes a breakdown of students by grade, enrollment trends, and preferred payment methods, offering a complete academic and administrative overview.
